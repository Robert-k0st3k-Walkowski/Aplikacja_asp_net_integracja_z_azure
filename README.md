# APLIKACJA ASP.NET + CHMURA MICROSOFT AZURE 💻

## UWAGA‼️
Aby aplikacja mogła zadziałać należy stworzyć sobie usługi azurowe: "Computer Vision", "Storage Accounts" oraz "Azure CosmosDB" i wrzucić do pliku "appsettings.json" klucze tych usług i wtedy wszystko powinno działać bez problemu! 😊

## Opis aplikacji 📜

Aplikacja ta jest prostą aplikacją webową, która realizuje operacje CRUD dla bazy danych związanej z samochodami.
<br><br>Głównym zamysłem aplikacji było to, żeby przećwiczyć łączenie się z usługami chmury "Microsoft Azure", ponieważ było to też motywem przedmiotu na studiach, w ramach którego ten projekt zrealizowałem.
